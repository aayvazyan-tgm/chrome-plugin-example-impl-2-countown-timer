# Product Overview

Chrome Extension Skeleton - A production-ready Chrome Extension Manifest V3 starter template.

## Purpose

Provides a clean, well-structured foundation for building Chrome extensions with:
- TypeScript-first development
- Automated testing (unit + E2E)
- CI/CD pipeline ready
- AI-friendly codebase with consistent patterns

## Current Features

- **Popup**: Extension popup UI (click extension icon)
- **Options Page**: Settings page with Chrome storage sync
- **Background Service Worker**: Handles extension lifecycle and messaging

## Target Users

Developers building Chrome extensions who want a modern, tested, and maintainable starting point.
